# Batch Tools

::: info 🚧 Roadmap
The following features are expected to be implemented in future versions.
:::

## 1. Batch Resize

- One-click scaling of dozens of screenshots to a specified width; height will scale proportionally.
- Quickly shrink a batch of high-definition images to 50% for easy sharing.

## 2. Batch Parameter Adjustment
*   **Supported Parameters**:
    *  Brightness, Contrast, Exposure.
    *  Saturation, Temperature, Tint.

## 3. Batch Watermarking
- Customize text content, font, opacity, and rotation. Supports importing Logo images and tile or fixed position modes.
