# Personalization and Behavior Settings

### General Settings

![General Settings Interface](/en/settings-general.webp)

*   **Auto-load images in same folder**: When enabled, opening an image will prompt TabPaint to scan for other images in the same directory.
*   **Default to View Mode**: When enabled, the software will automatically hide all toolbars upon startup, functioning as a pure image viewer.
*   **Theme**: Supports **Light**, **Dark**, or **Follow System**.
*   **Theme Color**: Provides 10 theme color options.
*   **Interface Language**: Currently supports **Simplified Chinese** and **English**.

---

### View Settings

![View Settings Interface](/en/settings-paint.webp)

*   **Mouse Wheel Function**: 
    *   **Zoom Image**: Wheel adjusts zoom.
    *   **Switch Image**: Wheel flips through pages.
*   **Background Display**:
    *   **Use Dark Background**: Canvas background around the image becomes dark gray (even in Light mode).
    *   **Show Transparency Grid**: When enabled, transparent areas of the image display a gray-and-white checkerboard; when disabled, a pure white background is shown.

### Drawing Settings

![Drawing Settings Interface](/en/settings-view.webp)

*   **Auto-popup Clipboard Images**: 
    *   When enabled, the TabPaint window will automatically bring itself to the front and activate once a new image is detected in the system clipboard.
*   **Selection Background Removal**: 
    *   The fill color for the empty area left after deleting a selection. Current options are **Transparent**, **Pure White**, or **Preserve Alpha**.
*   **Allow Delete Key to Delete Files**:
    *   To prevent accidental deletion, the direct file deletion function is disabled by default in Drawing mode. Used primarily for quickly deleting files during editing.

---

## Advanced Settings

![Advanced Settings Interface](/en/settings-advanced.webp)

*   **Resampling Algorithm**:
    *   Controls the algorithm used when scaling or saving images (e.g., Bilinear, Nearest Neighbor).
*   **Pixelation Threshold Settings**:
    *   Automatically toggles smooth anti-aliasing when the image zoom exceeds a certain multiplier (threshold). Thresholds can be adjusted separately for **View Mode** and **Drawing Mode**.
*   **No Popup for Workspace Reset**: Reduces popup notifications when resetting the workspace.
*   **Factory Reset**: Clears all cache and configurations, returning the software to its initial state.
