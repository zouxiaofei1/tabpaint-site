## View & Navigation


| Feature | Default Shortcut | Description |
| :--- | :--- | :--- |
| **Switch View/Edit Mode** | <kbd>Tab</kbd> | Toggle between "View Mode" and "Edit Mode" |
| **Previous Image** | <kbd>Left</kbd> | Switch to the previous image in the list |
| **Next Image** | <kbd>Right</kbd> | Switch to the next image in the list |
| **Full Screen** | <kbd>F11</kbd> | Toggle full screen |
| **Rotate Left** | <kbd>Ctrl</kbd> + <kbd>L</kbd> | Rotate image 90° counter-clockwise |
| **Rotate Right** | <kbd>Ctrl</kbd> + <kbd>R</kbd> | Rotate image 90° clockwise |
| **Flip Vertical** | <kbd>Ctrl</kbd> + <kbd>Alt</kbd> + <kbd>V</kbd> | Mirror image vertically |
| **Flip Horizontal** | <kbd>Ctrl</kbd> + <kbd>Alt</kbd> + <kbd>H</kbd> | Mirror image horizontally |

| **Text Tool** | <kbd>Ctrl</kbd> + <kbd>6</kbd> | Insert text box |

Apply color adjustments to the entire image.

| Feature | Default Shortcut | Description |
| :--- | :--- | :--- |
| **Brightness/Contrast/Exposure** | <kbd>Ctrl</kbd> + <kbd>Alt</kbd> + <kbd>R</kbd> | Adjust image light and contrast |

| **Resize** | <kbd>Ctrl</kbd> + <kbd>Alt</kbd> + <kbd>O</kbd> | Modify image resolution/dimensions |

## Modification and Reset

- **How to modify**: Click the recording box on the right of the corresponding feature and press the key combination on your keyboard (e.g., <kbd>Ctrl</kbd> + <kbd>Shift</kbd> + <kbd>A</kbd>).
- **How to reset**: Click **"Restore Default Shortcuts"** at the bottom of the page to revert to the initial state.
