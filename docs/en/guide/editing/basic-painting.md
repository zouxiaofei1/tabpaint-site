# Painting and Editing Tools

## 🎨 Basic Painting Tools

- Various brushes (**Pencil**, **Highlighter**, **Mosaic**, **Circle Pen**, **Brush & Airbrush**)
- Eraser
- Fill Tool
- Text Tool
- Selection Tool
- Rotate & Flip & Crop
- Utilities (**Screen Color Picker**, **Chroma Key**, **Trim Canvas Whitespace**, **Add Border**, **Ruler**)
- Image Adjustments (**Brightness**, **Contrast**, **Saturation**, **Exposure**, **Tint**, **Temperature**, **Black & White**, **Invert**, **Auto Levels**, **Canvas Size Adjustment**)

::: tip Transparent Brush Support
TabPaint supports opacity adjustment (Alpha channel), allowing you to layer strokes for rich textures.
:::
