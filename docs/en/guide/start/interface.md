# Interface Overview

TabPaint features a Mica material and borderless design.

## Main Interface
The main interface is divided into the Menu Bar, Image Bar, Canvas Area, and Status Bar.

![TabPaint Main Interface](/en/interface-main-dark.webp)

## Brushes and Toolbox
The brush settings are located at the top left, integrating tools such as Pencil, Eraser, Fill, Text, and Shapes.

![Brush Toolbar](/en/interface-tools.webp)

## Color Picker
TabPaint allows for quick selection of preset colors or precise selection using a full color wheel.

![Color Picker Interface](/en/interface-color-picker.webp)

## Settings Interface
Here you can customize TabPaint's behavior, including keyboard shortcut bindings, viewing/drawing preferences, language, and interface personalization options.

![Settings Interface](/en/settings-general.webp)

## Light Theme and Internationalization
TabPaint supports theme switching and multiple languages to adapt to different needs.

![Light Theme and English Interface](/en/interface-light-en.webp)
