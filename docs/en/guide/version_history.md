# Version History
 

## TabPaint 0.9.x

### v0.9.2
- **New**: Support for viewing and editing **.avif** format images.
- **New**: Welcome guide for first-time startup.
- **Fixed**: Resolved a crash issue with the right-click menu in specific scenarios.
- **Fixed**: Improved compatibility performance on Windows 10 systems.
- **Optimized**: Added MD5 checksum for AI model downloads.

### v0.9.0 - v0.9.1
- **Interface**: Full support for Dark Mode.
- **Interface**: Status bar layout reconstruction.
- **Language**: Added bilingual support (Chinese and English).
- **Distribution**: Launched a one-click installer.
- **Tools**: Color picker upgrade, supporting black/white quick colors and richer preset theme colors.
- **Optimized**: Improved memory management.

---

## TabPaint 0.8.x

### v0.8.6
- **Interaction**: Introduced **Responsive Toolbar** where tool icons automatically collapse based on window width.
- **Shortcuts**: Refined the shortcut settings system, supporting custom key bindings.
- **Integration**: Optimized the experience when working alongside screenshot software like Snipaste.

### v0.8.0 - v0.8.5
- **Core Features**: Officially separated **View Mode** and **Drawing Mode**, switching instantly via the <kbd>Tab</kbd> key.
  - **View Mode**: Pure and undisturbed, supports wheel zoom, left/right image switching, and EXIF info viewing.
  - **Drawing Mode**: Toolbar automatically appears for immediate editing.
- **AI Utilities**:
  - **One-click Background Removal**: Integrated AI model for intelligent background removal.
  - **OCR Text Recognition**: Extract text within a selection to the clipboard.
  - **Screen Color Picker**: Supports magnifier for precise color capturing anywhere on the screen.
  - **Smart Trim**: Automatically identifies and removes whitespace around the image.
- **Format Support**: Supports **GIF** playback.
- **Auxiliary Tools**: Added **Ruler** and **Grid Lines**.
- **Performance**: Deeply optimized for 4K/8K ultra-large images, significantly improving loading speeds and operation smoothness.
- **Experience**: Dragging a selection can now directly generate a new file or switch pages, making operations more intuitive.

---

## TabPaint 0.7.x 

- **Shape Tools**: Added vector drawing tools for rectangles, circles, arrows, rounded rectangles, etc.
- **Transform Tools**: Supports image **Rotation (Left/Right 90°)** and **Flipping**, with shortcut support.
- **Clipboard**: Added **Clipboard Listening Mode**; copied images automatically pop up for editing, multiplying screenshot workflow efficiency.
- **Interface**: UI style fully unified with Mica effects + Fluent UI for a more modern visual experience.
- **Interaction**: Supports <kbd>Space</kbd> + Drag to move the canvas (Hand mode).

---

## TabPaint 0.6.x

- **Multi-tab**: Refined the top **ImageBar**, supporting multi-image management and switching on the same screen.
- **Auto-cache**: Introduced the "Temporary File" concept; unsaved images are automatically cached, ensuring progress is not lost when closing the software, and eliminating the need for frequent "Save As".
- **Drag-and-Drop Interaction**:
  - Drag tab to desktop -> Copy file.
  - Drag external image into window -> Open or insert as a layer.
- **Editing Experience**: Optimized the Selection tool, supporting resizing, cropping, and copy-pasting.

---

## TabPaint 0.1 - 0.5

- **0.5**: Introduced Mica material effects and established borderless window design.
- **0.3**: Implemented basic ImageBar, supporting brush size, color adjustment, and undo/redo stack.
- **0.1**: Project initiation, completed basic Canvas framework.
