# 绘图与编辑工具

## 🎨 基本绘图工具


- 多种画笔 （**铅笔**，**荧光笔**，**马赛克**，**圆形笔**，**毛刷 & 喷枪**）
- 橡皮擦
- 颜色填充
- 文字工具
- 选区工具
- 旋转与翻转 & 裁剪
- 小工具（**屏幕取色器**，**色差抠图**，**裁切画布空白**，**一键加边框**，**标尺** ）
- 图像调整（**亮度**，**对比度**，**饱和度**，**曝光**，**色调**，**色温**，**黑白**，**反色**，**自动色阶**，**画布大小调节**）
::: tip 透明画刷支持
TabPaint 支持透明度调节（Alpha通道），你可以叠加笔触来创造丰富层次。
:::
