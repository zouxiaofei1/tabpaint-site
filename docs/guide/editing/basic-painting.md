# 绘图与编辑工具

TabPaint 不仅是一个轻量的看图工具，更是一个功能完备的绘图工坊。从简单的标注到复杂的修图，这里提供了你需要的所有基础工具。

## 🎨 基本绘图工具

工具栏位于顶部，包含了最常用的绘图操作。

### 多种画笔
TabPaint 提供了一套丰富的画笔库，满足不同场景的需求：
- **铅笔**：硬边缘，适合像素级修改或书写清晰文字。
- **荧光笔**：半透明效果，适合高亮文档重点。
- **马赛克**：快速模糊敏感信息（如人脸、密码）。
- **圆形笔**：标准的圆头画笔，绘图主力。
- **毛刷 & 喷枪**：带有纹理和边缘柔化效果，适合艺术创作或涂鸦。

::: tip 透明画刷支持
TabPaint 支持透明度调节（Alpha通道），你可以叠加笔触来创造丰富层次。
:::

### 橡皮擦
擦除画面内容。
- **右键技巧**：在使用橡皮擦时，支持右键调整大小。

### 颜色填充 (油漆桶)
一键填充封闭区域的颜色。支持容差设置，方便处理颜色不完全纯净的区域（如 JPG 噪点）。

### 选区工具
框选图像的一部分进行操作。
- **移动**：选中后直接拖拽像素。
- **变换**：配合 `Ctrl+T` 或菜单进行缩放、拉伸。
- **剪裁/复制**：配合 `Ctrl+X` / `Ctrl+C` 使用。

### 形状工具
快速绘制几何图形，支持 **矩形**、**圆形**、**直线**、**箭头** 等。
- **自动对齐**：按住 `Shift` 键可绘制正圆、正方形或水平/垂直线条。

### 旋转与翻转
- **旋转**：支持 90° 步进旋转 (快捷键 `Ctrl+L/R`) 或任意角度旋转。
- **翻转**：水平翻转或垂直翻转图像，适合制作镜像效果。

### 裁剪
重新定义画布范围。支持自由比例裁剪，或按固定比例（如 16:9, 4:3）裁剪。

## 🛠️ 实用小工具 (Gadgets)

在主菜单的“工具”栏中，TabPaint 集成了许多专为提高效率设计的微型工具。

### 屏幕取色器 & 颜色代码
不仅可以吸取当前画布的颜色，更支持 **屏幕全局取色**。
- **操作**：启动取色器后，鼠标移动到屏幕任何位置（包括 TabPaint 窗口外），点击即可吸取颜色。
- **复制颜色**：吸取后，可一键复制 Hex 代码 (如 `#FF5733`) 或 RGB 值到剪贴板，前端开发者的福音。

### 色差抠图 (魔棒)
简单的背景去除工具。点击背景色，根据色差范围自动选中并删除相似颜色的区域，快速实现透明背景。

### 智能图像处理
- **裁切画布空白**：自动检测图片四周的空白/透明像素并将其切除，让画布紧贴内容。截图文档时非常有用。
- **一键加边框**：快速给图片添加纯色边框或阴影效果，让截图在白色背景的文档中更具立体感。

### 标尺
显示像素标尺，辅助精准定位和测量尺寸。

## ✨ 图像调整与效果

如果需要对照片的整体色调进行修饰，TabPaint 提供了类 Lightroom 的调节滑块。

### 色彩调节
支持实时预览调节以下参数：
- **亮度 (Brightness)**
- **对比度 (Contrast)**
- **饱和度 (Saturation)**
- **曝光 (Exposure)**
- **色调 (Hue)**
- **色温 (Temperature)**

### 快速滤镜
- **黑白 (Grayscale)**：一键去色。
- **反色 (Invert)**：反转图像颜色（底片效果）。
- **自动色阶**：AI 自动分析直方图，优化图片的明暗分布，快速修复灰蒙蒙的照片。

### 调整画布大小
修改画布的分辨率（Resizing）。支持保持长宽比缩放，以及选择重采样算法（如邻近、双线性、双三次），保证缩放后的清晰度。

::: info 提示：非破坏性编辑
大部分色彩调整支持 **撤销 (Ctrl+Z)**。在 TabPaint 的缓存机制下，即使调节过度，也可以随时回退。
:::
