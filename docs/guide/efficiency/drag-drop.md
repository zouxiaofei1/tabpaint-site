# 拖拽与剪切板

通过极其丰富的拖拽支持，你可以将 TabPaint 作为图片处理的中转站，在桌面、Office 办公软件和 TabPaint 之间自由流转数据。

## 1.从 TabPaint 到外部

当你编辑完成或选定区域后，无需保存文件，直接通过拖拽即可输出。

### 拖拽选区 (Selection)
当你使用矩形选择工具或划定一个区域后，可以直接用鼠标按住选区内部进行拖拽：

- **拖拽至桌面/文件夹**：自动生成一个包含该选区内容的临时图像文件（`.png`）。
- **拖拽至 Word/PPT/微信**：直接将选区作为图片插入文档或发送给好友。

### 拖拽整张图片 (Thumbnail)
你可以直接拖拽底部的 **ImageBar（多标签缩略图栏）** 中的图片缩略图：

- **拖拽至桌面**：相当于“复制文件”到桌面。
- **拖拽至 Word/PPT**：直接插入整张图片。

::: tip 💡 单图模式的情况
当只打开一张图片时，你可以直接**拖拽程序左上角的 TabPaint Logo**，
它的作用等同于拖拽缩略图。
:::

## 2. 从外部到 TabPaint

TabPaint 根据你拖入位置的不同，智能判断你的意图。

### 拖拽文件进入窗口
当从桌面或资源管理器拖拽图片文件进入 TabPaint 时：

| 释放位置 | 效果 |
| :--- | :--- |
| **画布区域** | 将图片作为**新图层/选区**添加到当前画作中（适合拼图）。 |
| **ImageBar 区域** | 将图片作为**新标签页**打开（适合批量修图）。 |
| **标题栏区域** | 触发**切换工作区**，将该图片所在的文件夹作为当前工作目录加载。 |

### 剪切板监听 (Clipboard Monitor)
这是 TabPaint 处理截图的核心功能。

- **功能描述**：开启监听后，当你在其他软件（如浏览器、QQ、系统截图）中复制图片时，TabPaint 会自动将其作为一个**新的标签页**添加到后台。
- **应用场景**：连续截图 -> 自动收集 -> 批量处理 -> 批量保存。

## 3. 内部流转

TabPaint 支持在不同标签页之间“携带”数据。

### 跨标签页携带选区

1. 在图片 A 上创建一个**选区**。
2. 拖拽这个选区，移动到 ImageBar 的**图片 B 的缩略图**上。
3. **等待1s**。

**效果**：TabPaint 会自动切换到图片 B，并将刚才在图片 A 中选中的图像内容。
