# 多标签页与工作区管理


## 1. 图片加载方式

默认自动加载文件夹下的所有图片。
- 该功能可以在 `设置` 中关闭。

::: info `文件` 菜单打开模式的区别
- **打开**: 打开单张图片，加入图片栏中。
- **打开新工作区**: 关闭当前图片栏中的所有图片，加载目标文件夹下的所有内容。
:::

## 2. 文件与自动保存机制



-  每一次编辑都会被实时自动保存到系统的缓存目录：
  `C:\Users\用户名\AppData\Local\TabPaint` ，**下次启动时**自动恢复。按下 <kbd>Ctrl</kbd> + <kbd>S</kbd> 时，修改才会真正覆盖原始图片文件。
- 按下 <kbd>Ctrl</kbd> + <kbd>N</kbd> 或点击 ImageBar 上的 **`+`** 号可新建画布
::: info 💾 原图保护机制
在 TabPaint 中进行涂鸦、裁剪或调整时，所有的修改**不会立即写入原文件**。
:::

## 3. 批量管理工具


| 按钮功能 | 描述 |
| :--- | :--- |
| **💾 保存所有已编辑图片** | 将所有修改写入到原文件中。 |
| **❌ 关闭所有未编辑图片** | 清理工作区，只保留修改过的图片。 |
| **↩️ 放弃所有更改** | 清除缓存，重置为磁盘上的原始状态。 |
