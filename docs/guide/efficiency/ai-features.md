# AI 辅助功能


### 🪄 AI 智能抠图

画布右键 -> 小工具点击 `AI一键抠图`，等待片刻即可。


![AI 抠图效果对比](/gif-ai-workflow.gif)


---

### 📝 OCR 文字提取

框选图中文字(不框选则为全图识别)，画布右键 -> 小工具点击 `提取图中文字(OCR)`，识别结果将自动复制到剪贴板。
