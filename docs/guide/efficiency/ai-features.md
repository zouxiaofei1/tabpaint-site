# AI 辅助功能

TabPaint 通过集成轻量级的 AI 模型，完成抠图和文字提取。

## 🪄 AI 智能抠图

### 如何使用
1. 打开图片，或直接 <kbd>Ctrl</kbd> + <kbd>V</kbd> 粘贴剪贴板中的截图。
2. 在工具栏点击 **魔棒图标** (AI 抠图)。
3. 等待片刻，背景将自动变为透明。


::: tip ⚡ 相关技巧
配合 TabPaint 的拖拽特性，您可以实现：
**截图** → **自动进入 TabPaint** → **点击 AI 抠图** → **直接将结果拖入 PPT/Word**。
全过程无需保存文件，特别适合制作文档插图。
:::

## 📝 OCR 文字提取

### 如何使用

1. 框选图片中包含文字的区域(不框选则为全图识别)
2. 点击右键菜单的 **文本识别/OCR** 图标。
3. 识别结果将弹窗显示，并自动复制到剪贴板。
