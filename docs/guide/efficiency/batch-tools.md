# 批量工具

TabPaint即将支持基于多标签页的批量处理工作流。

::: info 🚧 功能预告 (Roadmap)
以下功能属于 **开发规划路线图** 的一部分，预计将在后续版本中陆续实装。
:::

## 1. 批量调整大小

基于 TabPaint 独特的“ImageBar”（类似 Notepad++ 的标签栏）设计，我们将引入批量尺寸控制。

*   **功能亮点**：
    *   **统一规格**：一键将选中的几十张截图统一缩放到指定的宽度（如 1920px），高度自动按比例缩放。
    *   **百分比缩放**：快速将一批高清大图缩小至 50% 以便分享。

## 2. 批量参数调整


*   **支持参数**：
    *   **基础光影**：亮度 (Brightness)、对比度 (Contrast)、曝光度 (Exposure)。
    *   **色彩平衡**：饱和度 (Saturation)、色温 (Temperature)、色调 (Tint)。

## 3. 批量添加水印


*   **功能规划**：
    *   **文字水印**：自定义文字内容、字体、透明度和旋转角度。
    *   **图片水印**：支持导入 Logo 图片，支持平铺或固定位置
