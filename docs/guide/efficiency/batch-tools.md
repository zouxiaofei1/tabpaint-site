# 批量工具

::: info 🚧 功能预告 (Roadmap)
以下功能预计将在后续版本中陆续实装。
:::

## 1. 批量调整大小

 - 一键将几十张截图统一缩放到指定的宽度，高度自动按比例缩放。
- 快速将一批高清大图缩小至 50% 以便分享。

## 2. 批量参数调整
*   **支持参数**：
    *  亮度 (Brightness)、对比度 (Contrast)、曝光度 (Exposure)。
    *  饱和度 (Saturation)、色温 (Temperature)、色调 (Tint)。

## 3. 批量添加水印
- 自定义文字内容、字体、透明度和旋转角度，支持导入 Logo 图片，支持平铺或固定位置
