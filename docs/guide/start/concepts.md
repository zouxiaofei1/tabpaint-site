# 核心概念

TabPaint 的设计理念是“介于画图和看图软件之间”。为了实现 10 秒内连续处理多张图片的极速工作流，你需要掌握以下四个核心交互概念。

## 1. 多标签页管理 (Notepad++ 模式)
TabPaint 不仅仅是一个单窗口画图工具，它引入了类似浏览器的多标签页机制。

*   **新建标签**：点击顶部菜单栏中间的 **`+`** 号按钮，或按下 `Ctrl + N`。
*   **切换标签**：点击标签标题即可切换不同的画布，处理多张截图时无需打开多个窗口。

## 2. Tab 键极速切换
这是 TabPaint 的灵魂功能。我们模糊了“看图”与“修图”的界限。

*   **按一下 <kbd>Tab</kbd>**：进入 **看图模式**。工具栏隐藏，界面清爽，适合沉浸式浏览。
*   **再按一下 <kbd>Tab</kbd>**：切回 **画图模式**。工具栏显现，随时可以对图片进行标注、裁剪或修改。

## 3. 拖拽即文件流
TabPaint 深度集成了 Windows 的拖拽特性，打破了软件与系统桌面的壁垒。

*   **保存/复制**：直接按住顶部的**标签页标题**，将其拖拽到桌面或文件夹中，即可快速保存当前画布为文件。
*   **插入文档**：如果是选区状态，直接拖拽选区进入 Word 或 PPT，即可直接插入图片。

## 4. 剪切板监听与 AI 协作
专为截图工作流优化。

*   **自动监听**：开启监听模式后，当你在其他软件（如微信、浏览器）中复制图片时，TabPaint 会自动将其作为新标签页打开。
*   **AI 抠图**：配合 AI 工具，复制进来的图片可以自动去除背景，让你专注于创作内容而非繁琐的抠图工作。
