# 核心概念

TabPaint 是介于画图和看图软件之间的画板。你可以先试试这些功能:

## 1. 多标签页管理
TabPaint 引入了多标签页机制。

*   **新建标签**：点击顶部菜单栏中间的 **`+`** 号按钮，或按下 `Ctrl + N`。
*   **切换标签**：点击标签标题或按下 `←` / ` →` 可切换不同的画布，处理多张截图时无需打开多个窗口。

## 2. Tab 键模式切换
这模糊了`看图`与`修图`的界限。

*   **按一下 <kbd>Tab</kbd>**：进入 **看图模式**，所有工具栏隐藏。
*   **再按一下 <kbd>Tab</kbd>**：切回 **画图模式**，s工具栏显现，随时可以对图片进行标注、裁剪或修改。

<!-- 制作一张动图 GIF，演示按下 Tab 键后界面的变化 -->
![Tab 键切换模式](/concepts-tabswitch.gif)

## 3. 无缝拖拽
通过拖拽，解决图片的传输难题。

*   **保存**：点击顶部的**图片标签**，拖拽到桌面或文件夹中，保存当前画布为文件。
*   **插入文档**：拖拽选区进入 Word 或 PPT，直接插入图片。

<!-- 制作一张动图 GIF，演示鼠标拖拽标签页标题到桌面生成文件的过程 -->
![拖拽复制画布](/concepts-drag1.gif)

## 4. 剪切板监听

*   **自动监听**：`Ctrl + Alt + P` 开启监听模式后，截图时 TabPaint 会将其作为新标签页打开。
*   **AI 抠图**：配合多种抠图工具，图片可以自动去除背景，无需保存，无需打开屏幕截图文件夹，无需复制粘贴。
