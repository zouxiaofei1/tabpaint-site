# 界面概览

TabPaint 采用了 Windows 11 原生的 Mica 材质与无边框设计。

## 主界面
主界面分为顶部菜单栏、顶部工具栏、画布区域和底部状态栏。界面默认随系统深色/浅色模式切换

![TabPaint 主界面](/interface-main-dark.png)

*   **顶部菜单栏**：包含文件、编辑、效果等传统菜单，以及标签页栏（ImageBar）。
*   **工具栏**：位于菜单下方，提供画笔、形状、裁剪等常用工具的快速访问。
*   **画布区域**：支持无限缩放和漫游的绘图区域。
*   **底部状态栏**：显示当前分辨率、缩放比例、光标坐标及缩放滑块。

## 画笔与工具箱
画笔工具栏位于顶部左侧，集成了铅笔、橡皮擦、填充桶、文本、形状等核心绘图工具。

![画笔工具栏](/interface-tools.png)


## 颜色选择器
TabPaint 提供了直观的色彩管理方案。您可以快速选择预设颜色，或调出完整色盘进行精确选色。支持吸管屏幕取色。

![颜色选择界面](/interface-color-picker.png)

## 设置界面
在这里您可以自定义 TabPaint 的行为，包括快捷键绑定、默认保存格式、AI 模型设置以及界面个性化选项。

![设置界面](/settings-general.png)

## 浅色主题与国际化
TabPaint支持色彩切换和多语言以适应不同需求

![浅色主题与英文界面](/interface-light-en.png)
