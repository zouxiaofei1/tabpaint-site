# 界面概览

TabPaint 采用了 Mica 材质与无边框设计。

## 主界面
主界面分为菜单栏、图片栏、画布区域和状态栏。

![TabPaint 主界面](/interface-main-dark.png)

## 画笔与工具箱
画笔位于顶部左侧，集成了铅笔、橡皮擦、填充、文本、形状等工具。

![画笔工具栏](/interface-tools.png)


## 颜色选择器
TabPaint 可以快速选择预设颜色，或使用完整色盘进行精确选色。

![颜色选择界面](/interface-color-picker.png)

## 设置界面
这里可以自定义 TabPaint 的行为，包括快捷键绑定、画图看图偏好、语言以及界面个性化选项。

![设置界面](/settings-general.png)

## 浅色主题与国际化
TabPaint 支持色彩切换，主题切换和多语言以适应不同需求。

![浅色主题与英文界面](/interface-light-en.png)
