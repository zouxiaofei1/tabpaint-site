# 个性化与行为设置
### 通用设置


![通用设置界面](/settings-general.png)

*   **自动加载同文件夹图片**:  开启后，打开一张图片时，TabPaint 会扫描同目录下的其他图片。
*   **默认进入看图模式**:开启此项后，软件启动时将自动隐藏所有工具栏，作为纯粹的看图工具使用。
*   **外观主题**: 支持 **浅色 (Light)**、**深色 (Dark)** 或 **跟随系统**。
*   **主题色**: 提供了 10 种主题色。
*   **界面语言**: 目前支持 **简体中文** 和 **English**。

---

### 看图设置

![看图设置界面](/settings-paint.png)

*   **鼠标滚轮功能**: 
    *   **缩放图片**: 滚轮缩放逻辑。
    *   **切换图片**: 滚轮用于翻页。
*   **背景显示**:
    *   **使用深色背景**: 图片周围的画布背景变为深灰色(浅色模式下也如此)。
    *   **显示透明度网格**: 开启后，图片的透明区域将显示灰白相间的棋盘格；关闭则显示纯白背景。

### 画图设置

![画图设置界面](/settings-view.png)

*   **剪贴板图片自动弹出**: 
    *   开启此项后，一旦系统剪贴板侦测到新图片，TabPaint 窗口会自动置顶并激活
*   **选区删除背景**: 
    *   删除选区后留下的空白区域填充的颜色，目前可以选择 **透明**、**纯白** 或 **保留透明度 (Alpha)**。
*   **允许 Delete 键删除文件**:
    *   为了防止误删，画图模式下默认禁用了直接删除文件的功能。主要用于修图时快速删除文件。

## 高级设置 (Advanced)

![高级设置界面](/settings-advanced.png)

*   **重采样算法 (Resampling)**:
    *   控制图片在缩放或保存时的算法（如双线性、邻近等）。
*   **像素化阈值设置**:
    *   当放大图片超过一定倍数（阈值）时，自动开关平滑抗锯齿,可以分别调整 **看图模式** 和 **画图模式** 的触发阈值。
*   **重置工作区时不弹窗提示**: 减少重置弹窗
*   **恢复出厂设置**: 清除所有缓存和配置，让软件回归初始状态。
